This page provides the complete classification data produced by our algorithm. Files are available in multiple formats (JSON, CSV, GAP) together with a compressed archive for bulk download.

Cite as: “The full classification datasets are available at https://rdelaby.github.io/atlas.html.” For archival stability, consider a Zenodo DOI tied to a GitHub release.

Contents

  • geometries.json — global list of geometries and metadata
  • incidence_matrices/ — matrices per instance (CSV/MTX)
  • gap/ — GAP representations
  • atlas.zip — everything in one archive

Summary

We classify geometries under constraints $(v, b, k, r, \ldots)$ with isomorphism filtering and proofs of completeness for specified parameter ranges. See the paper for the algorithmic details and proofs.
