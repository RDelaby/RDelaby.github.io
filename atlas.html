Incidence Geometry Atlas

Home Atlas

This page provides an atlas for flag transitive, residually connected, thin geometries admitting trialities and no dualities. For every group $G$ in the category, we provide the number of geometries, the size of the residues, the time it took to run it on a computer with specs X and for which groups of the category does the algorithm produce nothing. Group names are clickable and will link to the data in the form of triple of involutions. Add a read me with the formatting of the file. Add a file containing the algorithm. A readme on how to use the algorithm and the data is available here .

Here is a list of the families:

  1. Sporadic simple groups of order $\leq 10^{10}$.
  2. Classical groups, including $\mathrm{PGL}(2,q)$ and $\mathrm{PSL}(2,q)$ for $1 \leq q \leq 500$ a power of a prime.
  3. The alternating and symmetric groups $\mathrm{A}_n$ and $\mathrm{S}_n$ for $1 \leq n \leq 20$.
  4. Marston Conder's list of orientable hypermaps of genus $1$ to $101$.
  5. Miscs based on litterature.

    6. Cite as: “The full classification datasets are available at https://rdelaby.github.io/atlas.html.”
